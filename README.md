# Party
## Description:
這個想法是來自....?，沒有就是想開發

## Features:
 - 可以共享掉落物
 - 可以共享經驗
 - 防止隊員之間互相攻擊
 - 隊伍聊天

## Commands and Permissions:
- /party help -查詢命令及介紹
- /party gui -打開隊伍gui介面
- /party create name -創建一個隊伍
- /party join name -加入一個隊伍
- /party leave -離開一個隊伍
- /party list -查看有哪些隊伍
- @訊息(Massage) -隊伍聊天

## Install
1. Download jar file
2. Put it in plugins folder
3. Restart server
